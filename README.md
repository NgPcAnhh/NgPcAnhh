## 1) Nền tảng Hỗ trợ Phân tích đầu tư dựa vào Dữ liệu Cổ phiếu Việt Nam  
*(Đồ án tốt nghiệp)*

**Mô tả**  
Dự án tập trung thiết kế kiến trúc luồng dữ liệu phức tạp kết hợp giữa **Realtime** và **Scheduled**, từ đó xây dựng một nền tảng phân tích dữ liệu toàn diện cho thị trường chứng khoán Việt Nam.

**Đóng góp chính**
- **Xây dựng luồng dữ liệu chi tiết**
  - Thiết kế và triển khai các **Airflow DAGs** để thu thập, xử lý, lưu trữ và đồng bộ dữ liệu giữa nhiều tầng.
  - Xây dựng luồng **Kafka Realtime** để bắt dữ liệu giao dịch phục vụ hiển thị tức thời trên nền tảng.
- **Phát triển các tính năng CRUD trên nền tảng**
  - Xây dựng dashboard phân tích và nền tảng BI/kho truy vấn tích hợp trực tiếp trong hệ thống.
  - Xây dựng module quản lý người dùng và phân quyền truy cập.
- **Tích hợp các tính năng AI**
  - Phân tích cảm xúc thị trường từ tin tức bằng **PhoBERT** để chấm điểm và gán nhãn sau xử lý.
  - Phát triển **Text2SQL Chatbot** hỗ trợ truy vấn dữ liệu nhanh trong kho dữ liệu.

**Công nghệ**  
Apache Airflow, Apache Kafka, MinIO, PostgreSQL, FastAPI, Next.js, Docker

## 2) Triển khai Kiến trúc Data Lakehouse trong Xây dựng Nền tảng Phân tích Dữ liệu Thị trường Chứng khoán Việt Nam

**Mô tả**  
Dự án tập trung thiết kế và triển khai kiến trúc **Data Lakehouse** toàn diện, kết hợp **Modern Data Pipeline** với **BI Platform trực quan**, nhằm cung cấp giải pháp thu thập dữ liệu tự động, xử lý phân tán quy mô lớn, và tối ưu truy vấn phân tích cho thị trường chứng khoán Việt Nam.

**Đóng góp chính**
- **Xây dựng luồng dữ liệu chi tiết (Modern Data Lakehouse)**
  - Thiết kế luồng thu thập dữ liệu tự động và xử lý phân tán quy mô lớn.
  - Tổ chức lưu trữ tối ưu cho OLAP, chuẩn hóa và đồng bộ dữ liệu hiệu năng cao giữa các tầng.
  - Ứng dụng kiến trúc Lakehouse để nâng cao tính nhất quán và khả năng mở rộng của toàn hệ thống.
- **Xây dựng nền tảng BI & Dashboard động (CRUD & Visualizations)**
  - Phát triển bộ công cụ truy vấn và phân tích trực quan.
  - Cho phép người dùng cấu hình chi tiết biểu đồ bằng JavaScript với **Apache ECharts**.
  - Thiết kế giao diện động hỗ trợ kéo-thả và sắp xếp bố cục biểu đồ linh hoạt trên Dashboard.
- **Tích hợp tính năng AI (AI Vibecoding Agent)**
  - Xây dựng trợ lý AI Agent hỗ trợ tạo và điều chỉnh mã nguồn biểu đồ từ ngôn ngữ tự nhiên.
  - Tối ưu hệ thống prompt để cải thiện chất lượng sinh mã biểu đồ.
  - Hướng tới trải nghiệm **low-code / no-code** trong xây dựng báo cáo phân tích.

**Công nghệ**  
Airflow, Kafka, Apache Spark, MinIO, Apache Iceberg, ClickHouse, PostgreSQL, FastAPI, Next.js, Docker

**Source**  
[stock-lakehouse](https://github.com/NgPcAnhh/stock-lakehouse.git)
